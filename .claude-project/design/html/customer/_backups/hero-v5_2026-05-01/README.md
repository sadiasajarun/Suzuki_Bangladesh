## How to reuse

### Reapply to this project

```bash
# From customer/ directory
cp _backups/hero-v5_2026-05-01/hero.css css/sections/hero.css
cp _backups/hero-v5_2026-05-01/hero.js  js/sections/hero.js
```

Then make sure `index.html` references both:

```html
<link rel="stylesheet" href="css/sections/hero.css" />
<script defer src="js/sections/hero.js"></script>
```

And the hero `<section id="hero" class="hero">` markup is in place (see git history of `index.html` at the time this backup was made).

### Reuse on another project

Copy the entire `customer/` directory and:
1. Make sure GSAP and Tailwind CDN scripts are loaded in `<head>`
2. Make sure the four bike images are present at `assets/images/bikes/{gixxer-sf-250.webp, gsx-r150.webp, access-125.jpg}`
3. Make sure the Suzuki logo PNG is at `assets/images/logo/suzuki-logo.png`

Or use the standalone `Hero-v5_original.html` — it's fully self-contained.
